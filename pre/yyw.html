Retrieval strategy

ML-Enhanced FAQ Retrieval

After simple intents are filtered out, Nova narrows the search space before ranking candidate answers.

1

Category Classification

TF-IDF features + Logistic Regression predict the likely FAQ category.

2

Category-specific Retrieval

Search only within the predicted category to reduce noise.

3

Similarity Ranking

TF-IDF + Cosine Similarity rank Top-K FAQ candidates.

4

Confidence Assessment

Similarity score and thresholds decide how direct the answer should be.

Response Strategy

HighReturn the direct verified FAQ answer.
MediumShow the top 3 related questions for student confirmation.
LowUse fallback or ask for clarification.
Chatbot Output · HTML UI

FAQ answer · predicted category · confidence score · top 3 related questions · recent chats / session history
