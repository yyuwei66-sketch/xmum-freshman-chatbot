# 厦马资讯Q&A——第4期（新生篇）

欢迎大家关注公众号：大马口袋！

还没进群的同学可以扫下面二维码或者后台留言拉你进厦大马校聊天资讯群，里面可以随意聊天，还会为大家分享各种马来西亚的资讯和及时解答大家校内外各种问题。

Q：团员档案和个人档案需不需要带到学校？

A：

因为学校没有团支部，所以厦马不能入党，也不能存放团员档案。大家的团员档案只能寄存在自己的高中学校或者户籍所在地的社区。

个人档案的话，往年是需要带到马校上交给学校的，毕业后学校会把个人档案寄回户籍所在地或其他地方。具体详情等待学校通知。

Q：学校课程如何分类？

A：本科课程分类：

1.Major Core Course专业必修课：每门课程3-5学分

本专业的基础科目，是学科地基，很重要！

2.Common Core Course公共必修课：马来语，马来历史等，每门课程2学分。

体验马来西亚文化及提升个人素养，与本专业无关。

3.Major Elective Course（ME）专业选修课：每门课程3学分

本专业的分支科目，可以根据读研方向选择适合的课程，很重要！

4.General Elective Course（GE）通识选修课：每门课程2-3学分

烘焙，运动，鉴赏，编程，商业和各类专业课都有，根据自己兴趣去选择的课程，可以与本专业无关。

学校本科每年有三个学期：9月份，2月份和4月份。9月份和4月份两个学期为大学期，有十五周的课程，以专业必修课，专业选修课和公共必修课为主。2月份小学期有八周的课程，主要上通识选修课。

GPA（Grade Point Average）平均绩点：本学期每个科目所获得的绩点乘以科目学分相加之后的总和除以本学期学分总和。

CGPA（CumulativeGrade Point Average）累计平均绩点：所有学期的加权平均分。

GPA和CGPA的满分都是4分。申请研究生主要看CGPA。

Q：学校如何选课？是不是需要抢课？

A：学校只有ME和GE课需要选课。ME课基本不用抢课，因为我们学校各专业人数较少。GE课是整届同学一起选课，有可能需要抢。ME和GE的每门课都有固定的人数上限，选课流程主要分为三轮：

第一轮：任意选课，大概给三天时间选择要选修的课。此轮每门课报名可以突破人数上限，按照各自喜欢的课程报名就行。你选的课如果第一轮结束之后没超人数上限，你就成功选上了。如果超了，则随机选出相对应的课程人数上限。剩下没被选上的同学，将进行第二轮选课。这轮不需要抢课。

第二轮：先到先得，持续三天左右。第一轮课没选上的同学，需要抢第一轮没录满的课程，开放时间一到就可以去抢，先到先得。

第三轮：开学第一周为试课周，如果有不满意的课，可以在第一周的周一到周五更换课程。一般这周课程不用签到。

开学前一到两个星期会发邮件告诉大家选课流程，开放选课系统。大一基本上不用选课，详情留意本专业信息。

Q：学校怎么发奖学金？

A：

学校奖学金：每学年结束，会根据成绩排名发放不同百分比的奖学金。会用于下一学年的学费减免上。中国同学拿到的奖学金大概2000+马币。

学校本地统考生第一学年奖学金：入学那年会根据统考学科成绩获得多少个A进行多少百分比的学分减免。一个A，可以减免5%，依次递增，如果全部A（一共10个）的话，学费全免。

Q：学校什么时候可以转专业？

A：大一上学期不允许转专业。转同大类专业，可以第一学期结束就转。不同大类之间转专业需要一学年以后才能转。若转会计和金融，需要考试：在大一4月份学期的5月份左右考英语和数学。转其他专业根据专业要求和拿合格的语言成绩去申请即可。

而且在校期间只能转一次专业，转专业一般就没有了保研资格。我们学校也暂未开放修双学位。

Q：学校转专业是否会延迟毕业？

A：转专业会不会延迟毕业的问题，要根据转的专业是属于同大类还是非同大类和转后专业的学业完成时间而定。

比如：物理转数学，属于转同大类专业，大一第一学期结束就能申请转专业。数学专业在不挂科的情况下三年时间就能完成所有课程，但第四年结束拿毕业证书。也就是说物理转数学专业顺利完成学业只需花3年➕一个大学期的时间就可以完成，并不会延迟毕业（晚于四年毕业的才叫做延迟毕业）。

小tips：每个学期都有固定的学分上限，所修的课程总学分不能超过每学期上限。但是老师给各位同学排课的时候，会预留部分学分给大家补课或者重修，所以如果想转专业后按时完成本科学习，可以让自己的Academic Advisor（学术顾问老师，每个同学都会安排一位）帮你们排课表。如果每学期能多安排1门课，就能加快毕业进度。但是有可能要加学费而且要看加修的课程和原有的课程有没有时间上冲突。

以上都是根据学姐学长个人经验给大家总结出来的，可以作为一个参考。关于学校的问题以学校官方回答为准。
